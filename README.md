# ECG-Recording-Using-Microcontrollers 
This program is a microcontroller program that I have built to recording the ECG signal. The devices that I use is [Mikromedia 4 for STM32F4](https://mikroe.com/mikromedia-4-for-stm32-capacitive), medical instrumentation [ECG Click](https://www.mikroe.com/ecg-click) and serial communication [USB UART Click](https://www.mikroe.com/usb-uart-click) from Mikroe. Then the Multiparameter Simulator MS400 is the signal generator of Ischemic heart signal such as ST-wave Elevation and Depression signal. I choose the Elevation and depression signal with 0.2mV, 0.3mV and 0.4mV (positive is the elevation value and negative is depression value). Recording did 6 minutes for every signals. MS400 has a 3 electrodes that attach on 3.5 jack of ECG Click. Those recording was use a 250 Hz of sampling frequency. The recorded signal is then saved in text format. After that, the saved signal is then processing using a Discrete Wavelet Transform. Further processing of the ECG signal you can gind out in [here](https://github.com/AchmadFachturrohman/Machine-Learning-Based-Myocardial-Ischemia-Classification/tree/Discrete-Wavelet-Transform).
