# Real-time ECG Recording 
This program is a microcontroller program that I have built to recording the ECG signal. The devices that I use is Mikromedia 4 for STM32F4 and ECG Click from Mikroe. Then the Multiparameter Simulator MS400 is the signal generator of Ischemic heart signal such as ST-wave Elevation and Depression signal. I choose the Elevation and depression signal with 0.2mV, 0.3mV and 0.4mV (positive is the elevation value and negative is depression value). Recording did 6 minutes for every signals. MS400 has a 3 electrodes that attach on 3.5 jack of ECG Click. The recorded signal is then saved in text format. After that, the saved signal is then processing using a Discrete Wavelet Transform. The discrete wavelet trnasform session you can gind out in "Discrete Wavelet Transform".
